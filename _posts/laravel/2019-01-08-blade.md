### 변수 출력
{% raw %}
* {{ $foo }}
* {{ $foo['name'] }}
{% endraw %}

### Sanitizing
{% raw %}
* {{ $foo }} = \<?php echo htmlentities($foo); ?>
* {!! $foo !!} = \<?php echo $foo; ?>
{% endraw %}
### 조건문
```php
@if ($num > 5)
5보다 큽니다.
@elseif($num == 5)
5와 같습니다. 
@else
5보다 작습니다.
@endif
```  

### 반복문
```php
@foreach($tasks as $task)
    {% raw %}{{ $task['name'] }}{% endraw %}
@empty
    값이 없네요...
@endforeach
```

### 레이아웃과 상속
* resources/views/layouts/master.blade.php

```html
<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="utf-8" />
    <title> @yield('title') </title>
</head>
<body>
    @yield('content')
</body>
</html>
```
* welcome.blade.php

```html
@extends('layouts.master')

@section('title')
welcome
@endsection

@section('content')
hello world
@endsection
```

### 조각 뷰 삽입
* resources/views/partials/footer.blade.php

```html
<footer>
    <p>footer 입니다.</p>
</footer>
```
* welcome.blade.php

```html
@extends('layouts.master')

@section('content')
    @include('partials.footer')
@section
```
