### 람다 변환 2
```kotlin
class Person(val name: String, val age: Int)

val people = listOf(Person("kjg", 11), Person("leh", 22))

people.maxBy ({ p: Person -> p.age})

// 맨 뒤의 인자가 람다 식이면 괄호 밖으로 빼낼 수 있음
people.maxBy() { p: Person -> p.age}

// ()에 인자가 없으면 ()를 없앨 수 있음
people.maxBy { it.age }

people.maxBy(Person::age)
```
