### @Query 애노테이션
> 메소드이름으로 표현하기 힘들 경우

- JPQL
```java
@Query(value = "SELECT c FROM Comment AS c")
List<Comment> findByTitleContains(String keyword);
```

- SQL
```java
@Query(value = "SELECT c FROM Comment AS c", nativeQuery = true)
List<Comment> findByTitleContains(String keyword);
```
