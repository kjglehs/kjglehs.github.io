### ORM 패러다임 불일치
> 객체를 릴레이션에 맵핑하면서 발생하는 문제와 해결책

#### 밀도 문제
- 객체 : 다양한 크기, 커스텀 타입을 만들 수 있음
- 릴레이션 : 기본 데이터 타입만 가능(UDT는 비추)

#### 서브타입 문제
- 객체 : 상속 구조, 다형성
- 릴레이션 : 방법이 없거나 표준기술이 아님

#### 식별성 문제
- 객체 : 레퍼런스 동일성(==), 인스턴스 동일성(equals 메소드)
- 릴레이션 : 주키(Primary key)
